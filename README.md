# Proyecto de Robótica - Grupo 1 2026

Este repositorio contiene la organización, documentación y avance del proyecto de robótica del grupo.

## Estructura del repositorio

- `maquina-de-estados-grupo1/`: contiene el desarrollo del bloque de máquina de estados.
- `README.md`: descripción general del repositorio.
- `.gitignore`: configuración para ignorar archivos innecesarios.

## Contenido del bloque principal

Dentro de `maquina-de-estados-grupo1/` se encuentran:

- `src/`: archivos fuente
- `docs/`: documentación
- `images/`: diagramas, capturas o imágenes de apoyo

## Observación

Este repositorio funciona como respaldo público en GitHub y como espacio de organización del trabajo colaborativo.
